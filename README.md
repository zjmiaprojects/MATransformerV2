# MATransformerV2

The codes for the work "Lightweight Multi-Stage Aggregation Transformer for Robust Medical Image Segmentation"
![image](https://github.com/zjmiaprojects/MATransformerV2/blob/main/GraphicalAbstract.jpg)

data
-
COVID-19 CT Dataset is available at https://medicalsegmentation.com/covid19/,

MoNu Dataset is available at https://monuseg.grand-challenge.org/Data/,

EM Dataset is available at https://www.epfl.ch/labs/cvlab/data/data-em/.

Checkpoints
-
In the Transformer branch, we utilize the pretraining parameters of [MobileViT](https://arxiv.org/abs/2110.02178) on ImageNet1k, which can be downloaded from https://github.com/apple/ml-cvnets
